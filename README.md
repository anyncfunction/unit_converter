# ⚡ 单位换算器

一款基于 Python Tkinter开发的图形界面单位换算工具，支持多种单位类型换算，带有暗黑霓虹风格界面。

## ✨ 特性

- 🎨 **炫酷界面** - 暗黑主题 + 霓虹紫粉青配色
- 🔄 **7种换算类型** - 长度、重量、温度、体积、面积、速度、时间
- 📜 **历史记录** - 自动保存换算记录，关闭后数据保留
- ⌨️ **快捷操作** - 支持回车键快速转换
- 🖥️ **跨平台** - Windows/Linux/Mac 均可运行

## 🚀 快速开始

### 运行程序

```bash
python main.py
```

### 依赖

- Python 3.7+
- 无需额外安装第三方库

## 📋 支持的单位

| 类型 | 单位 |
|:---|:---|
| 长度 | 米、千米、厘米、毫米、英里、码、英尺、英寸... |
| 重量 | 千克、克、毫克、吨、磅、盎司、斤、两 |
| 温度 | 摄氏度、华氏度、开尔文 |
| 体积 | 升、毫升、立方米、加仑、夸脱、品脱... |
| 面积 | 平方米、平方千米、公顷、亩、英亩... |
| 速度 | 米/秒、千米/时、英里/时、节、马赫 |
| 时间 | 秒、毫秒、分钟、小时、天、周、月、年 |

## 📁 项目结构

```
unit_converter/
├── main.py                 # 程序入口
├── gui.py                  # GUI界面
├── converter.py            # 换算核心逻辑
├── converter_history.json  # 历史记录管理
└── README.md               # 说明文档
```

## 🎯 使用方法

1. 选择换算类型（长度/重量/温度等）
2. 输入要转换的数值
3. 选择"从"和"到"的单位
4. 点击"开始转换"或按回车
5. 结果显示在下方，历史记录自动保存到右侧

## 🐛 常见问题

**Q: 运行报错怎么办？**
> 确保 Python 版本 >= 3.7

**Q: 历史记录在哪里？**
> 程序会自动创建 `converter_history.json` 文件保存历史

**Q: 怎么清空历史？**
> 点击右侧的"清空"按钮
